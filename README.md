# Modern Robotics CPP

**Author**: Jingkun Liu

A Modern C++ Package that provides the functions accompanying the book [*Modern Robotics:
Mechanics, Planning, and Control*](http://modernrobotics.org) (Kevin Lynch
and Frank Park, Cambridge University Press 2017)

## Target Platform

| Environment | Value |
| ----------- | ----- |
| Platform | Ubuntu 22.04 LTS |
| Kernel | 6.9.3-76060903-generic |
| CMake | 3.22.1 |
| g++ | 11.4.0 |
| Armadillo | 10.8.2 |

## Covered Topics

1. Rigit Mody Motion: includes all functions that computes the rigid body transformations, that conducts all transformations on rotation and homogenous transformation matrix.
